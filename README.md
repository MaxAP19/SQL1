# SQL1
Basic constructions of SQL

## Описание:
Проект посвящён изучению основ реляционной модели данных и языка SQL. Работа велась с базой данных PostgreSQL, включающей таблицы pizzeria, person, menu, person_visits и person_order, с акцентом на получение и обработку данных с помощью базовых SQL-конструкций.

## Этапы выполнения проекта:
- Изучение принципов реляционной модели и стандартов SQL.
- Создание и заполнение таблиц для пиццерий, клиентов, меню, посещений и заказов.
- Написание SELECT-запросов с фильтрацией, сортировкой и вычисляемыми полями.
- Практика внутренних запросов (subquery) и условных выражений.
- Применение сортировки, работы с диапазонами данных и форматирования строк.
- Последовательное выполнение упражнений от простого извлечения данных до более сложных запросов без использования JOIN и IN.

## Освоенные компетенции:
- Написание базовых SQL-запросов с фильтрацией и сортировкой.
- Форматирование и вычисление колонок в результатах запросов.
- Использование внутренних подзапросов в SELECT и FROM.
- Применение условных выражений и проверка значений.
- Работа с интервалами и диапазонами данных.
- Выполнение SQL-запросов с учётом ограничений (без JOIN и IN).
- Понимание структуры реляционных таблиц и практическое получение данных в PostgreSQL.
